Chibi Productivity RPG App

A productivity app that I randomly decided to build one day as I wanted an app we=here we can level up our progress like those heroines in otome game but there were none so I just decided to make it myself. It's still a work in progress though. In this there are 8 categories that we can choose which are, study, fitness, creativity, career, social, fashion, expenses, sprituality, and chores. In this app, there is a Quest log where User can put in what they've done or havent done. For example, study chapter 1 of computer science. If the User succeeded in their goal, they will click on the green button where their rank on the the study category will go up. However, if they failed and click on the red button, their category point for the Study category will be deducted. There are also timer function, and function where we can view our history and deletes it if we want. Other than that, there is also a dressing room function where User can dress up their avatar, for now there are only two dress options, however I am planning to increase the dressing up options, such as adding hair options, adding more props, etc. All of the drawings used are all drawn by me.
